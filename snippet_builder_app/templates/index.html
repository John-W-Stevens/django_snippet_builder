Snippet Builder

This compact full-stack application was built with the Django 2.2 web development framework. The following technologies were used in the development of this app: Python3, Django 2.2, HTML, CSS (Bootstrap 4), and JavaScript (jQuery). This app was deployed on an Amazon Web Services (AWS) cloud server using NGINX and Gunicorn.

How it works:

Select your code editor and programming language, enter a description and tab trigger, write (or paste) your snippet code in the Ace editor, click copy to clipboard, open a new snippet file in your code editor, replace the entire content of that file with your snippet, save it, and start using your snippet.

Why I built it:

I built this app because both the text editors I use (Sublime Text and VS Code) don't have GUI's for creating snippets and I didn't find their respective snippet templates to be particularly intuitive. I wanted an easy way to create snippets quickly.
